28mm Base Decal Template - Instructions

Setup for Printing

  • Paper: Use A4 transfer/decal paper (water-slide or heat-transfer)
  • Print Settings:
    • Set margins to: 0.2" (5mm) on all sides
    • Paper size: A4 (210 × 297mm)
    • Scale: 100% (no scaling)
    • Color: Black ink recommended
  • Font Size Reference:
    • Small (9pt) = ~3.2mm height - for tiny bases
    • Medium (11pt) = ~3.9mm height - standard for 28mm
    • Large (13pt) = ~4.6mm height - for important names

Application Instructions

  • Let decals dry completely (varies by type, typically 24-48 hours)
  • Cut out each decal with a sharp knife, leaving minimal white border
  • Soak in water for 30-60 seconds (water-slide)
  • OR apply directly and smooth with water (heat-transfer)
  • Position on base rim and smooth out air bubbles
  • Seal with matte or gloss varnish once dry

Tips

  • Print on plain paper first to test alignment
  • Edit the HTML to add your own names in the decal divs
  • Adjust class to "size-small", "size-medium", or "size-large" as needed
  • The grid arranges 4 columns for optimal page usage
  • The gang logo (3×3mm @ 300DPI) can be added above or below names
  • Use negative letter-spacing to keep names compact for tiny bases
